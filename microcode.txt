MAR = memory address register
MDR = memory data register

======================= Control Signals =======================
MAR_WRITE => Write to MAR from address bus

MDR_READ  => Read to data bus from MDR
MDR_WRITE => Write to MDR from data bus

MEM_READ  => Read to MDR from memory address in MAR
MEM_WRITE => Write to memory address in MAR from MDR

REG_READ  => Read to register from data bus
REG_WRITE => Write to data bus from register
